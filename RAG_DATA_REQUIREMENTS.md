# RAG知识库数据需求分析

## 🎯 问题分析
用户问题："我现在有5000元预算，请告诉我是现在购买iPhone15 Pro好呢，还是等双十一降价后购买iPhone16呢？"

要回答这个问题，Agent需要以下数据：

## 📋 必需的数据类型

### 1. 产品基本信息 (products)
- 产品名称、型号、发布日期
- 当前价格、原价
- 配置参数、规格
- 产品图片、描述

### 2. 价格历史数据 (price_history)
- 历史价格变化
- 促销活动记录
- 季节性价格趋势
- 双十一等大促价格

### 3. 产品发布时间线 (product_timeline)
- 历代产品发布时间
- 产品生命周期规律
- 发布间隔规律
- 降价周期

### 4. 市场分析数据 (market_analysis)
- 市场需求趋势
- 消费者偏好
- 竞品分析
- 行业报告

### 5. 购买建议数据 (purchase_recommendations)
- 历史购买建议
- 用户反馈数据
- 购买时机分析
- 预算规划建议

### 6. 促销活动数据 (promotions)
- 双十一、618等大促活动
- 优惠券信息
- 降价幅度统计
- 活动时间表

## 🗄️ 数据来源建议

### 1. 官方数据源
- Apple官网产品信息
- 官方价格历史
- 发布公告

### 2. 电商平台数据
- 京东、淘宝、拼多多
- 价格监控数据
- 用户评价数据

### 3. 第三方数据源
- 科技媒体报告
- 行业分析文章
- 价格追踪网站

### 4. 社交媒体数据
- 微博、小红书讨论
- 用户购买体验
- 产品评价

每个数据表都需要标准化格式，便于Agent理解和分析。
